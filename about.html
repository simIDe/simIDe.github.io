About

I did a thesis on exoskeletons at the interface between biomechanics, neuroscience and robotics. I was particularly excited about using statistics, data processing, and visualization tools that I could manipulate during this journey. My interest in these domains led me to the job of data engineer that I currently practice. My project is to continue to develop my skills in machine learning, deep learning and web technologies to become a full-stack data scientist.
